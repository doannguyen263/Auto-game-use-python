# Game Automation Tool

Tool tự động hóa game trên emulator với giao diện đồ họa.

## Cài đặt

```bash
python -m pip install -r requirements.txt
```

## Sử dụng

### GUI Mode
```bash
python main.py
```

### CLI Mode
```bash
python main.py --no-gui --game <game_name> --task <task_name>
```

### Build EXE
```bash
python build_exe.py
```
